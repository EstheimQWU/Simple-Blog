## 概要
这个项目的目的是手把手教学快速搭建一个基于springboot与VueJS的个人博客站点。
## 技术栈
### 本篇内容
+ 前端数据：VueJS2.0
+ 前端UI: ELEMENT-UI, AMAZEUI
+ 后端：Springboot 1.4.0 Released
+ 数据库：MariaDB 10.2
+ 运行环境：centos 7.3
### 额外内容
+ 容器：docker
+ 图片存储：fastfds
